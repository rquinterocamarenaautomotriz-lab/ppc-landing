# PPC Landing

Landing estática para PPC — Poliurea Protección Confiable.

## Archivos principales
- `index.html`: landing completa con CSS y JS embebidos.
- `assets/images`: imágenes optimizadas.
- `assets/videos`: videos comprimidos para web.

## Subida a GitHub
Subir todo el contenido de esta carpeta al repositorio `ppc-landing`.
Netlify actualizará automáticamente el sitio conectado.
